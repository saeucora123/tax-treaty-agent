# Internal History Archive

This directory is a local holding area for non-public repo context.

It is used to park items that are useful for later audit, cleanup, or recovery, but should not live in the public GitHub reading path.

Typical contents include:

- chat exports
- private Codex working notes
- local run logs
- temporary downloaded source files used during verification

Subdirectories under this archive are intentionally ignored by Git so they stay available locally without being part of the public repository snapshot.
