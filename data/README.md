# Data Layout

- `source_documents/`: governed source inputs used to build treaty artifacts.
- `treaties/`: runtime treaty datasets consumed by the backend.
- `onboarding/`: shadow re-onboarding manifests, OECD baseline references, reviewer-edited source payloads, and promotion workdirs for the offline treaty compiler.
- `evals/`: replay packs and evaluation fixtures for regression runs.
- `raw_documents/`: research-oriented raw materials and parser inputs, not runtime datasets.
